# Keerthana MP Portfolio

Modern personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Notes

 - Place your profile photo at `public/profile.jpg` (the app loads `/profile.jpg` in the hero).
 - Replace `public/resume.pdf` with your actual resume to enable the "Download Resume" button.
- Update social links in `src/components/Contact.tsx` and `src/components/Navbar.tsx` if needed.
